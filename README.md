# Blog CanedoDev

## Getting Started

Primeiro comece dando um start no Servidor:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Digite [http://localhost:3000](http://localhost:3000) no seu navegador preferido para ver o resultado.
